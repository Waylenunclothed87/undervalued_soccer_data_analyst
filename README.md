## 📊 Understanding the Data

The application calculates a value score for each player. A high value score suggests the player performs better than their current market price implies. 

The system uses historical data from platforms like Transfermarkt. It accounts for factors such as goals, assists, minutes played, and defensive contributions. The model processes this information to generate a prediction of what the player should be worth in the current market.

## 📝 Frequently Asked Questions

What if I cannot find a specific player? 
The tool focuses on major professional leagues. If a player is in a smaller league or does not have sufficient public data, the model may exclude them from the primary search results.

Does this software predict future transfers?
No. This tool identifies current market inefficiencies. It tells you which players currently offer the best performance for their cost. It does not predict future move decisions by clubs.

Can I run this on a Mac?
This version supports Windows only. 

How does the 5-fold cross-validation work?
The software splits the data into five parts. It trains the model on four parts and tests it on the fifth. It repeats this process for every part to ensure the model remains stable and accurate across different types of players. 

Is there a cost to use this?
The software is free to download and use for personal scouting purposes.
